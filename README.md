# Frontend README

This folder contains the frontend application source code for the ViverSorvete project.

## Structure

```
frontend/
├── docs/
│   ├── TASKS.md              # Task tracking
│   └── tasks/                # Individual task files
├── src/                     # Application source code
└── README.md                # This file
```

## Technology Stack

- **Framework**: React
- **Language**: TypeScript
- **Build Tool**: Vite
- **Styling**: CSS Modules

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
