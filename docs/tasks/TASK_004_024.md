# TASK: Implement Linear Login Redirect and Tenant Selection Flow

---

## DESCRIPTION

Make the login UX deterministic for both platform and tenant users.

- Summary (1 sentence):
  Implement a linear frontend auth flow where the login response directly determines whether the user enters a platform page or a tenant selection step.
- Steps (max 5):
  1. Redirect platform users directly to the platform home page
  2. Redirect single-tenant users directly to the tenant home page
  3. Redirect multi-tenant users to tenant selection
  4. Use explicit auth response scope/context to choose the next route
  5. Remove route logic that depends on implicit legacy assumptions
